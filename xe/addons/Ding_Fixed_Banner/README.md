# Ding_Fixed_Banner (Addon)
![ding_fixed_banner_logo](http://download.xpressengine.com/xedownload/app/22753633/thumbnails/md.jpg "ding_fixed_banner_logo")


###개요
브라우저 상단, 하단에 고정된 배너를 출력해주는 위젯입니다. 
내부 콘텐츠를 HTML, CSS으로 작성할수 있어서 다양한 배너를 만드는데 용이합니다.
쇼핑몰에서 가장 많이 사용합니다.

###설치 및 설정 방법
1. 위젯을 해당 지정된 경로에 설치합니다.
2. 자신의 홈페이지에 맞게 설정합니다.

###기능과 특징

[+ 1.5.2]

1. 배너 쿠키에 아무 값도 입력하지 않았을 때 배너가 출력되지 않는 오류가 해결되었습니다.

[+ 1.5.1]

1. IE8에서 식별자 오류가 발생하는 문제가 해결되었습니다. 

[+ 1.5.0]

1. 모바일에서 작동을 확인했습니다.
2. 애니메이션을 선택 적으로 적용 가능 하도록 하였습니다.
3. 특정 태그에 배너를 끼워넣을 수 있는 기능이 추가 되었습니다.

[+ 1.4.0]

1. 배너와 같은 포지션을 갖는 내비게이션과 레이아웃 호환을 위해서 내비게이션을 선택하고 css top을 수정할 수 있도록 추가되었습니다. 후 처리 애니메이션도 추가되었습니다.

[+ 1.3.7] [+ 1.3.8]

1. 배너와 같은 포지션을 갖는 네비게이션이 설계되어 있는 레이아웃 호환을 위해 top을 조정할 수 있도록 수정되었습니다.
2. 기본으로 제공하는 버튼의 위치를 수정할 수 있도록 변경되었습니다.
3. 닫기 버튼을 눌렀을 때 가끔 쿠키가 적용되지 않는 문제가 수정되었습니다.

[+ 1.3.5] [+ 1.3.6]

1. HTML을 입력할 때 "를 \"이렇게 써야 하는 문제점을 " 그대로 입력해도 되게 수정하였습니다.
2. HTML을 입력할 때 엔터를 입력하면 안 되게 변경되었습니다. 공백은 허용됩니다.

[+ 1.3.4]

1. '위'를 설정하고 높이를 변경했을 때 변경되지 않던 문제를 수정했습니다.
2. 자바스크립트를 캐싱 할 수 있도록 변경되었습니다.
3. HTML 코드를 입력하는 부분의 사용방법이 변경되었습니다.
4. 소스 코드 최적화.
 
[+ 1.3] [+ 1.3.1] [+ 1.3.2] 

1. 타 자료와 충돌방지.
2. 소스코드 최적화.

[+ 1.2]

1. Cookie를 통해서 닫기 버튼을 누를 시 지연 '일(day)'를 추가할 수 있게 되었습니다.
2. 소스코드 최적화.

[+ 1.1]

1. 기본 닫기 SVG 버튼이 추가되었습니다. (활성화, 비활성화 존재합니다.) 
2. 기본 닫기 버튼의 색상을 설정할 수 있도록 되었습니다.
3. conf가 수정되었습니다.
4. 설치 경로가 ./addons/fixed_notice 에서 ./addons/Ding_Fixed_Banner 으로 변경 되었습니다. 

[+ 1.0]

1. HTML을 입력할 수 있는 입력기에 텍스트를 입력해도 되지만 태그를 입력해도 됩니다.
2. 해당 입력한 태그에 클래스를 지정하고 CSS를 입력할 수 있습니다.
